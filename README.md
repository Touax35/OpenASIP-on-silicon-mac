# OpenASIP-on-silicon-mac

OpenASIP (openasip.org) on MacOS M1

# INSTALLATION

- install Docker  
  https://desktop.docker.com/mac/main/arm64/Docker.dmg
- install XQuartz  
  https://www.xquartz.org/index.html
- Build and launch Docker image and container for OpenASIP  
  ./openasip

# LAUNCH

./openasip
